# Jeebly Dashboard

**Jeebly Dashboard** is a large-scale **multi-restaurant admin platform** designed to manage an online food ordering system similar to **Talabat** or **Uber Eats**.

The platform acts as a central system that contains multiple restaurants, where **each restaurant has its own dedicated dashboard** to manage its data independently.

The dashboard is built using **Angular** and is designed to integrate with backend APIs developed using **.NET Core**.

---

## 🧩 Platform Overview

Jeebly is designed as a scalable system that supports:

- Multiple restaurants under one platform
- Separate dashboards for each restaurant
- Central admin control
- Role-based access control (RBAC)
- Advanced reporting and analytics

---

## 🏪 Core Features

### 🔹 Restaurants Management
- Add and manage multiple restaurants
- Each restaurant has:
  - Its own categories
  - Its own products
  - Its own orders
  - Independent dashboard view

### 🔹 Categories & Products
- Category management per restaurant
- Product listing and management
- Product availability & pricing structure

### 🔹 Orders Management
- View and manage incoming orders
- Order status tracking
- Restaurant-specific order handling

### 🔹 Users & Accounts
- Authentication & authorization system
- Multiple user roles (Admin, Restaurant Owner, Staff, etc.)
- Role-based permissions for accessing features

### 🔹 Reports & Analytics
- Orders reports
- Sales summaries
- Restaurant performance metrics
- System-wide analytics for super admins
